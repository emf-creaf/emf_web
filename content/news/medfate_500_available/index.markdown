`medfate` R package, with allows simulating forest functioning and dynamics of forest stands has been updated to version `5.0.0` in CRAN.  

In this new version, the most relevant improvement is computational, since the
package has undergone an internal code re-structuration to replace Rcpp
structures by C++ structures, resulting in substantial increases in speed.

In addition, `medfate` now incorporates the simulation of organic matter
decomposition (dead wood, litter and soil organic carbon), following an
adaptation of the DAYCENT model.

Read all changes made in this version
[here](https://emf-creaf.github.io/medfate/news/index.html#medfate-500).  
Read about `medfate` examples of usage
[here](https://emf-creaf.github.io/medfate/index.html).  
A complete description of model design and formulation can be found
[here](https://emf-creaf.github.io/medfatebook/index.html).
